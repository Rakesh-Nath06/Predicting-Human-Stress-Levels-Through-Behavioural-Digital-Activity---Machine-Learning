## 🎯 Objective

- Predict stress levels (Low / Medium / High)
- Compare multiple machine learning models
- Identify the most accurate model

---

## 📊 Dataset

- 25,450 records  
- 8 features  
- Balanced dataset  

### Features:
- Age  
- Gender  
- Sleep Duration  
- Quality of Sleep  
- Physical Activity Level  
- Work Hours  
- Screen Time  

---

## ⚙️ Workflow

1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Encoding & Scaling  
4. Train-Test Split  
5. Model Training  
6. Model Evaluation  
7. Deployment using Gradio  

---

## 🤖 Models Used

- Logistic Regression  
- Random Forest  
- Gradient Boosting  
- Support Vector Machine (SVM)  

---

## 📈 Results

- Random Forest → 73.06%
- 🏆 Gradient Boosting → 74.16%  (Best Model)
- Logistic Regression → 69.87%  
- SVM → 74.02%  

---

## 🔍 Key Insights

- Sleep duration is a strong predictor of stress  
- High screen time correlates with increased stress  
- Work hours significantly influence stress levels
